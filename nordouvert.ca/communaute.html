Guides

Faire pression pour les données ouvertes

  • Code for America offre un guide pour vous aider à militer pour l'ouverture de votre gouvernement local. On y trouve aussi des échantillons de présentation et de dépliants à présenter aux représentants élus.

Adopter les données ouvertes

  • Le Open Data Field Guide de Socrata est “un guide complet permettant de vous assurer que votre programme de données ouvertes est efficace pour vous et pour les citoyens”.
  • Code for America décrit les étapes pour introduire des politiques sur les données ouverteset pour créer une culture d’ouverture gouvernementale de l’intérieur du gouvernement.
  • Le Open Data Policy Guidelines de la Sunlight Foundation répond à la question “Que peut ou devrait accomplir une politique sur les données ouvertes?” par 32 clauses.
  • Le Open Data Commons de l’Open Knowledge Foundation est un coffre à outils légaux pour vous aider à fournir et à utiliser les données ouvertes.
  • Le Municipal Lobbying Data Guidebook de la Sunlight Foundation aide les décideurs locaux à mieux divulguer l’information sur le lobbyisme.

Utiliser les données ouvertes

  • La School of Data offre des cours en ligne et une formation pratique, dresse l’inventaire des outils en ligne et des ressources et publie un manuel.
  • Le Spending Data Handbook de l’Open Knowledge Foundation est pour tous ceux qui souhaitent utiliser et comprendre les budgets gouvernementaux et les données sur les dépenses dans le cadre de leur travail.
  • Code Academy offre des cours pour vous aider à utiliser les données ouvertes disponibles à travers les API de la Sunlight Foundation.
  • Le Data Journalism Handbook est un livre en ligne pour tous ceux qui s’intéresse au journalisme de données. Jetez aussi un coup d’oeil au guide Open Spending pour les journalistes pour comprendre comment est alloué l'argent.

Déployer des applications civiques

  • mySociety a une série de guides portant sur les façons de créer un site web permettant de suivre le gouvernement, de signaler des problèmes ou demander de l'information (plus de détails ici).
  • Code for America créé des guides brefs sur les façons les plus faciles de redéployer des applications civiques open source.

En apprendre davantage sur les données ouvertes

  • Le Open Data Handbook de l’Open Knowledge Foundation introduit les aspects légaux, sociaux et techniques des données ouvertes.
  • Les 10 principes pour ouvrir l'information gouvernementale décrit de façon succincte points importants des données ouvertes.
  • Posez des questions ou offrez des réponses au Open Data Stack Exchange.
